# Agent User Management System (PHP + SQLite)

This project is a simple **User Management System** built using:

- PHP (Backend)
- SQLite (Database)
- HTML (Views)
- CSS (User Interface)

It includes role-based user creation for the following roles:

- Admin
- Agent
- Player
- Club Manager
